# Controle de Presença no Escritório

Aplicação Streamlit para controle de presença no escritório, férias e feriados.

## Funcionalidades

- Visualização de calendário mensal
- Marcação de dias de presença no escritório
- Marcação de férias
- Feriados automáticos para 2026
- Contador de dias de presença por mês
- Persistência de dados em JSON

## Como usar

1. Instale as dependências:
```bash
pip install streamlit
```

2. Execute a aplicação:
```bash
streamlit run controle_escritorio.py
```

3. Acesse no navegador (normalmente abre automaticamente em http://localhost:8501)

## Arquivo de dados

Os dados são salvos automaticamente em `presencas_calendario.json`.
